### 2. 과대/과소 추정 변화

v3는 "보수적 추정"과 "패키징 과다 추가 방지"를 강조한 결과:
- 과대추정: 37.4% → **9.4%** (대폭 감소)
- 과소추정: 61.7% → **90.2%** (대폭 증가)

**문제점**: v3가 너무 보수적으로 추정하여 과소추정이 증가함.

## 결론

### v3의 성과
1. **스타굿즈 카테고리 대폭 개선** (73.2% → 52.6%)
   - v2가 포토카드/팬시 상품을 과대추정하던 문제 해결
   - 인형/피규어 과대추정 문제도 개선
   
2. **과대추정 문제 해결** (37.4% → 9.4%)
   - "최소 박스 크기" 규칙 완화
   - 패키징 여유 축소

### v3의 한계
1. **전체 MAE 후퇴** (47.7% → 51.9%)
   - v2에서 잘 동작하던 카테고리들이 악화됨
   
2. **과소추정 급증** (61.7% → 90.2%)
   - 너무 보수적인 추정으로 인해 대부분의 상품을 과소추정
   
3. **의류/패션/화장품 악화**
   - 이 카테고리들은 v2의 패키징 규칙이 더 적합했음

### 개선 방향 제안

**v4 프롬프트 방향:**
1. **하이브리드 접근**: 카테고리별로 다른 규칙 적용
   - 스타굿즈/인형: v3 방식 (보수적)
   - 의류/화장품: v2 방식 (패키징 여유)
   
2. **과소추정 방지 규칙 추가**
   - "실제 무게보다 낮게 추정하지 말 것" 명시
   
3. **카테고리별 무게 가이드 세분화**
   - v2의 볼링가방, 로봇 등 성공 사례 유지
   - v3의 포토카드, 인형 성공 사례 유지
